### 1. Hq_Vendor_00001 : 구매의뢰요청관리 (본사)
- **화면명**: 본사 구매의뢰요청관리
- **메뉴 경로**: 본사 > 매입발주 > 매입관리 > 구매의뢰요청관리
- **주요 기능**: 
  - 본사 관리자가 가맹점/매장으로부터 올라온 구매의뢰 내역을 종합적으로 모니터링하고 가맹점 신규 의뢰를 대행 등록하거나 편집합니다.
  - 임시저장, 구매의뢰 요청 등의 상태 관리가 가능합니다.
- **검증 및 조치 완료**:
  - [x] Oracle $\rightarrow$ PostgreSQL 쿼리 호환성 확인 및 Null-Safety 캐스팅 적용
  - [x] 품목 비고란 한글 글자 크기(50Byte) 초과 시 브라우저 차단 및 알림 메시지 제한 조건 적용
  - [x] E2E Playwright 검증 성공
  - **완료 여부**: ✅ **완료 (조치 완료)**

### 4. Hq_Vendor_00004 : 검수관리 (본사)
- **화면명**: 본사 검수관리
- **메뉴 경로**: 본사 > 매입발주 > 매입관리 > 검수관리
- **주요 기능**:
  - 본사 관리자 권한으로 매장별/거래처별 발주 및 납품 처리된 건에 대하여 최종 검수 수량을 변경 및 입력하여 저장하고, **검수 확정** 또는 **검수 취소** 상태 천이(`PROC_FG`: '1' $\leftrightarrow$ '2')를 수행하는 화면입니다.
- **검증 및 조치 완료**:
  - [x] 빈 문자열 입력 시 PostgreSQL 캐스팅 Null-Safety 쿼리 적용 (`saveCheckDetail`)
  - [x] 발주번호 검색창 최대 **12자리**, 전표번호 검색창 최대 **4자리** 길이 제한 속성(`maxlength`) 적용
  - [x] E2E Playwright 검수 저장/확정/취소 및 Bootbox 모달 자동 검증 성공
  - **완료 여부**: ✅ **완료 (조치 완료)**
